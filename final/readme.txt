The folder contains final project in Physical Design for Nanometer Ics (NTU)
at 2019-2 semeter, with the following files:  
-	Course Problem File: ICCAD20_ProblemD.pdf
-	Coding         File: readin.ipynb, DBreadin.ipynb, Basicmodel.ipynb,
			     DB_Basicmodel.ipynb, DB_PCAmodel.ipynb 
-       Report         File: report.pdf
-	GivenData    Folder: /base_nom_0p8v, /base_nom_0p9v, /base_nom_1p0v
-	InputData Container: /8v_case[1~5], /9v_case[1~5], DB_8v_case[1~5]

Coding problem is ICCAD-Contest 2020's Problem D--Fast Timing Model Estimation for new PVT.
We can run readin.ipynb, DB_readin.ipynb as "Data Crawler" to get input data for model.
We can run Basicmodel.ipynb, DB_Basicmodel.ipynb, DB_PCAmodel.ipynb to get the result.

-       作業報告檔: report.pdf
-       給定測資檔: /base_nom_0p8v, /base_nom_0p9v, /base_nom_1p0v
-	輸入測資夾: /8v_case[1~5], /9v_case[1~5], DB_8v_case[1~5]

程式問題是ICCAD-2020競賽的問題D--Fast Timing Model Estimation for new PVT.
