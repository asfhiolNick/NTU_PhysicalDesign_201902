The folder contains 3rd homework in Physical Design for Nanometer Ics (NTU)
at 2019-2 semeter, with the following files:  
-	Course Problem File: prog3_placement.pdf
-	Coding       Folder: /src
-       Report         File: report.pdf
-       Executive      File: /bin/place
-	InputData    Folder: /input_pa3
-       Support      Folder: /lib

Coding problem is to place a group of modules and  with minimum cost(wire-length+frame-size).
We can use Makefile to generate .place in /bin, and operate it.
